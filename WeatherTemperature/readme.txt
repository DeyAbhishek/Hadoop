1800.csv has some real weather data for the year 1800. 
It contains the min temperature, max temperature for each and every day of 1800 for certain weather stations.

The MinTemperature.py python script returns the min temperature for the weather stations in the entire 1800.

How To Run:

python MinTemperature.py 1800.csv

OR

python MinTemperature.py 1800.csv > result.txt
